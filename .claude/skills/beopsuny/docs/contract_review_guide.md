# 계약서 검토 보조 가이드

> 법순이 + Claude 하이브리드 계약서 검토 워크플로우

## 목적 및 범위

### 지원 범위

| 계약서 유형 | 제공 기능 |
|------------|----------|
| **한국어 계약서** | 조항 → 한국 법령 레퍼런스, 효력 이슈 경고 |
| **영문 계약서** | 위 기능 + 법률 용어 영한 번역, 뉘앙스 차이 설명 |

### 법순이가 하는 것
- 계약서 조항 → 관련 **한국 법령** 레퍼런스 제공
- (영문 계약서) 법률 용어의 **영한 번역** + 한국법상 뉘앙스 차이 설명
- 한국 강행규정, 약관규제법 등 **효력 이슈** 경고

### 법순이가 하지 않는 것
- 계약서 자동 분석/위험도 점수
- 수정안 자동 생성
- 법률 자문 또는 승소 가능성 예측

> **주의**: 본 가이드는 초벌 검토 보조용입니다. 최종 검토는 반드시 변호사와 상담하세요.

## Phase 0: 횡단 이슈 체크 ⭐

> 조항별 검토 **전에** 계약 전체 맥락에서 적용되는 법령 먼저 확인

### 0.1 국제거래 세금 (해외법인 상대방)

**체크 질문**: 상대방이 해외 법인인가?

해외법인과 거래 시 조항별 법령 외에 **세법**이 전체 계약에 적용됩니다.

| 이슈 | 설명 | 확인 법령 |
|------|------|----------|
| **원천징수** | 외국법인 지급액에서 원천세 공제 | 법인세법 제93조, 제98조 |
| **조세조약** | 국가별 세율 다름 (미국 11%, 일본 10% 등) | 조세특례제한법, 개별 조약 |
| **전자용역 부가세** | SaaS 등 전자용역은 해외사업자 납부 의무 | 부가가치세법 제53조의2 |

**법순이 명령어:**
```bash
# 원천징수 관련
fetch_law.py exact "법인세법"
# → 제93조 (국내원천소득), 제98조 (원천징수)

# 조세조약 적용 시
fetch_law.py exact "조세특례제한법"
# → 제28조 (조세조약에 따른 원천징수세액 경감)

# 전자용역 부가세 (SaaS 등)
fetch_law.py exact "부가가치세법"
# → 제53조의2 (전자적 용역을 공급하는 국외사업자의 사업자등록 특례)
```

**핵심 체크**: 상대방 소재지 → 조세조약 → 세금 부담 조항 ("net of taxes")

### 0.2 하도급법 (대기업-중소기업 용역)

**체크 질문**: 원사업자-수급사업자 관계인가?

대기업이 중소기업에 용역 발주 시 **하도급법** 강제 적용됩니다.

| 적용 기준 | 내용 |
|----------|------|
| **원사업자** | 중소기업 아닌 사업자 (대기업, 중견기업) |
| **수급사업자** | 중소기업 (연 매출 1,000억 이하 등) |
| **대상 용역** | 용역위탁: 건설/수리/시공 제외한 용역 |
| **금액 기준** | 연간 위탁금액 10억원 이상 시 서면 의무 |

**법순이 명령어:**
```bash
# 하도급법 본문
fetch_law.py exact "하도급거래 공정화에 관한 법률"
# → 제2조 (정의), 제3조 (서면 교부 의무)
# → 제13조 (하도급대금의 결정), 제13조의2 (감액 금지)

# 하도급 관련 행정규칙 (실무 기준)
fetch_law.py search "하도급" --type admrul
# → 표준하도급계약서, 하도급 분쟁 고시 등

# 공정위 제재 동향
fetch_policy.py rss ftc --keyword 하도급
```

**핵심 체크**: 기업 규모 → 서면 의무 → 대금/기술자료 조항 위반 여부

### 0.3 기타 횡단 이슈

| 이슈 | 적용 조건 | 확인 법령 |
|------|----------|----------|
| **표준계약서 의무** | 대리점, 가맹점, 콘텐츠 계약 | 대리점법, 가맹사업법, 콘텐츠산업진흥법 |
| **외국환 신고** | 1만불 이상 대외지급 | 외국환거래법 제18조 |
| **수출입 규제** | 전략물자, 이중용도품목 | 대외무역법, 전략물자관리원 |

```bash
# 대리점/가맹 계약 시
fetch_law.py exact "대리점거래의 공정화에 관한 법률"
fetch_law.py exact "가맹사업거래의 공정화에 관한 법률"

# 외국환 신고 (고액 거래 시)
fetch_law.py exact "외국환거래법"
```

## Phase 1: 계약서 구조 파악

### 1.1 계약 유형 식별

| 유형 | 주요 확인 법령 | 법순이 명령어 |
|------|---------------|--------------|
| SaaS/라이선스 | 저작권법, 개인정보보호법 | `fetch_law.py exact "저작권법"` |
| 용역/서비스 | 민법 도급, 위임 | `fetch_law.py exact "민법"` |
| 공급/구매 | 상법 매매, 하도급법 | `fetch_law.py exact "하도급거래 공정화에 관한 법률"` |
| 고용/위탁 | 근로기준법, 파견법 | `fetch_law.py exact "근로기준법"` |
| 투자/M&A | 상법 회사편, 자본시장법 | `fetch_law.py exact "상법"` |
| NDA/비밀유지 | 부정경쟁방지법 | `fetch_law.py exact "부정경쟁방지 및 영업비밀보호에 관한 법률"` |

### 1.2 당사자 역할 확인

- **갑/을 관계** 파악: 우리 쪽이 고객인지 공급자인지

```bash
# 약관규제법 핵심 조항 확인
fetch_law.py exact "약관의 규제에 관한 법률"
```

## Phase 2: 조항별 검토

### 2.1 Boilerplate 조항 (일반 조항)

| 조항명 | 한국법 고려사항 | 위험도 |
|--------|----------------|--------|
| **Governing Law** | 한국 소비자는 한국법 강행규정 적용 (국제사법 제27조) | 중 |
| **Dispute Resolution** | 중재합의 시 중재법 제8조 서면 요건 | 중 |
| **Entire Agreement** | 한국법상 계약 전 협상 내용도 해석에 참작 가능 | 하 |
| **Severability** | 민법 제137조 일부무효 원칙과 유사 | 하 |
| **Assignment** | 채권양도 시 민법 제450조 통지 요건 | 하 |

### 2.2 Risk Allocation 조항 (위험 배분)

| 조항명 | 한국법 고려사항 | 위험도 |
|--------|----------------|--------|
| **Indemnification** | 약관규제법 제7조: 부당한 면책 무효 | **상** |
| **Limitation of Liability** | 고의/중과실 면책 불가 (약관규제법 제7조) | **상** |
| **Consequential Damages 배제** | 특별손해 예견 시 무효 주장 가능 (민법 제393조) | 중 |
| **Force Majeure** | 불가항력 범위 한국법과 차이 가능 | 중 |

### 2.3 IP & Data 조항 (지식재산/데이터)

| 조항명 | 한국법 고려사항 | 위험도 |
|--------|----------------|--------|
| **Work for Hire** | 저작권법 제9조 요건 엄격 (외주 불포함) | **상** |
| **IP Assignment** | 저작인격권 양도 불가 (저작권법 제14조) | 중 |
| **Data Processing** | 개인정보보호법 제28조의8 국외이전 요건 | **상** |
| **License Grant** | 독점/비독점 범위, 2차적저작물 권리 확인 | 중 |

### 2.4 Employment 관련 조항

| 조항명 | 한국법 고려사항 | 위험도 |
|--------|----------------|--------|
| **Non-Compete** | 헌법 제15조 직업선택자유 vs 영업비밀보호 | **상** |
| **Non-Solicitation** | 범위 과도 시 공서양속 위반 가능 | 중 |
| **Invention Assignment** | 발명진흥법 제10-15조 직무발명 보상 필수 | 중 |

## 용어 번역 시 주의사항

### 영미법 특유 개념

| 영문 | 직역 문제 | 올바른 이해 |
|------|----------|-------------|
| Consideration | 고려사항 (X) | 약인 - 한국법에 없는 개념 |
| Warranty vs Representation | 둘 다 보증? | Warranty=보증, Rep=진술 (효과 다름) |
| Specific Performance | 특정 이행? | 강제이행 (금전배상 아닌 현물이행) |
| Punitive Damages | 징벌적 손해배상 | 한국법상 원칙 불인정 |

### 한국법 효력 차이

| 영문 조항 | 한국법 효력 |
|----------|------------|
| Waiver of moral rights | **무효** (저작인격권 일신전속성) |
| Unlimited indemnity | **무효 가능** (약관규제법) |
| Non-compete > 3년 | **무효 가능** (공서양속) |
| Punitive damages | **집행 제한** (한국 판례) |
