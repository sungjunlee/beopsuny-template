# 해외 진출 시 한국법 확인 가이드

해외직접투자, 전략물자 수출, 국제조세 등 해외 진출 시 **한국법에서 확인해야 할 사항**을 안내합니다.

> **주의**: 이 가이드는 "어디서 확인해야 하는지" 안내입니다.
> 구체적 기준(금액, 비율, 절차)은 **반드시 최신 법령/기관에서 확인**하세요.
> 외국법(미국법, EU법 등) 관련 사항은 해당 국가 전문가와 상담하세요.

---

## 1. 해외직접투자 신고

외국법인 설립, 지분 취득, 금전 대여 등을 통해 해외 사업에 참여하는 경우.

> **참고**: 증권사를 통한 일반 해외상장주식 투자는 별도 신고 불필요.
> 해외직접투자는 외국법인 지분 10% 이상 또는 임원 파견 등 경영 참여 시 해당.

### 확인할 법령

```bash
fetch_law.py exact "외국환거래법"
fetch_law.py search "외국환거래규정" --type admrul
fetch_law.py search "해외직접투자" --type admrul
```

### 신고 기관

- **지정거래외국환은행** (시중은행 외환 창구)

### 주요 확인 사항

- 신고 대상 여부 (지분율 10% 기준, 임원 파견 등 예외)
- 사전신고 vs 사후신고
- 사후 보고 의무 (외화증권취득보고서, 연간사업실적보고서)

### 참고 사이트

- [한국은행 외환거래 심사](https://www.bok.or.kr/portal/main/contents.do?menuNo=200405)
- [은행연합회 해외직접투자 안내](https://exchange.kfb.or.kr/page/exchange05.php)

## 2. 전략물자 수출

국제 수출통제체제에 따라 제한되는 물품/기술을 수출하는 경우.

### 확인할 법령

```bash
fetch_law.py exact "대외무역법"
fetch_law.py search "전략물자" --type admrul
```

### 판정/허가 기관

- **무역안보관리원 (KOSTI)** - 전문판정
- **전략물자관리시스템** (yestrade.go.kr) - 자가판정/신청
- 산업통상자원부 - 수출허가

### 판정 방식

| 구분 | 자가판정 | 전문판정 |
|------|---------|---------|
| 판정 주체 | 기업이 직접 | 무역안보관리원 (KOSTI) |
| 절차 | yestrade.go.kr 온라인 | 전문가 검토 + 심의회 |
| 적합한 경우 | 명확한 품목 | 판단 어려운 품목 |

### 주요 확인 사항

- 전략물자 해당 여부 판정
- 수출허가 유형 (개별, 포괄)
- 최종사용자 확인
- 우려 용도 전용 가능성 (상황허가)

### 참고 사이트

- [전략물자관리시스템 yestrade.go.kr](https://yestrade.go.kr/)
- [무역안보관리원 KOSTI](https://www.kosti.or.kr/)
- [찾기쉬운 생활법령 - 전략물자 수출](https://easylaw.go.kr/CSP/CnpClsMain.laf?popMenu=ov&csmSeq=577&ccfNo=5&cciNo=2&cnpClsNo=1)

## 3. 이전가격/국제조세

특수관계자(해외 본사, 자회사 등)와의 거래가 있는 경우.

### 확인할 법령

```bash
fetch_law.py exact "국제조세조정에 관한 법률"
fetch_law.py search "이전가격" --type admrul
```

### 관할 기관

- **국세청**

### 주요 확인 사항

- 정상가격 원칙 (arm's length principle)
- 이전가격 문서화 의무
- 국가별 보고서 (CbCR) 제출 대상 여부

### 참고

- OECD 이전가격 가이드라인
- 국세청 국제조세 안내

## 6. 서류 인증 (아포스티유)

### 확인할 법령

```bash
fetch_law.py search "아포스티유" --type admrul
fetch_law.py exact "재외공관 공증법"
```

### 발급 기관

- **외교부 재외동포청** - 아포스티유 확인서 발급

- 대상 문서 유형별 절차 차이

### 참고 사이트

## 7. 분쟁해결 및 국제중재

해외 거래 계약에서 분쟁해결 조항 설계 시 고려사항.

### 확인할 법령

```bash
fetch_law.py exact "중재법"
fetch_law.py exact "국제사법"
```

### 주요 확인 사항

- 준거법 선택 (governing law)
- 분쟁해결 방식: 소송 vs 중재 vs 조정
- 중재지 (seat of arbitration) 선택
- 중재판정 집행 가능성 (뉴욕협약 가입국 여부)

### 주요 국제중재기관

| 기관 | 약칭 | 특징 | 웹사이트 |
|------|------|------|----------|
| 국제상업회의소 | ICC | 세계 최대, 복잡한 대형 분쟁 | [iccwbo.org](https://iccwbo.org/dispute-resolution/) |
| 싱가포르 국제중재센터 | SIAC | 아시아 대표, 빠른 절차 | [siac.org.sg](https://www.siac.org.sg/) |
| 홍콩 국제중재센터 | HKIAC | 중국 관련 분쟁에 강점 | [hkiac.org](https://www.hkiac.org/) |
| 대한상사중재원 국제중재 | KCAB | 한국 기업에 유리, 비용 효율 | [kcabinternational.or.kr](https://www.kcabinternational.or.kr/) |

### 참고 사이트

- [대한상사중재원](https://www.kcab.or.kr/)

## 8. 산업기술 유출방지

국가핵심기술 또는 산업기술을 해외로 이전하거나, 기술 보유 기업이 해외 M&A 대상이 되는 경우.

> **주의**: 2024년 12월 법 개정으로 규제가 대폭 강화되었습니다.
> 국가핵심기술 보유 여부를 모르고 위반 시에도 처벌 대상이 될 수 있습니다.

### 확인할 법령

```bash
fetch_law.py exact "산업기술의 유출방지 및 보호에 관한 법률"
fetch_law.py search "국가핵심기술" --type admrul
```

### 관할 기관

- **산업통상자원부** - 국가핵심기술 수출 승인/신고
- **국가정보원** - 산업기술 유출 수사

### 주요 확인 사항

- 국가핵심기술 보유 여부 판정 (13개 분야 76개 기술 지정)
- 수출/해외인수합병 시 **사전 승인 또는 신고** 필요
- 기술 이전 계약, 합작투자 시 검토
- 위반 시 징역 및 벌금 (2024년 개정으로 대폭 상향)

### 참고 사이트

- [산업통상자원부 산업기술보호](https://www.motie.go.kr/)
- [국가법령정보센터 - 산업기술유출방지법](https://www.law.go.kr/법령/산업기술의유출방지및보호에관한법률)

## 9. 개인정보 국외이전

해외 클라우드 서비스 이용, 해외 본사/자회사와 개인정보 공유 등의 경우.

> **참고**: 2023년 개정으로 동의 외 국외이전 방법이 다양화되었습니다.

### 확인할 법령

```bash
fetch_law.py exact "개인정보 보호법"
fetch_law.py search "국외이전" --type admrul
```

### 관할 기관

- **개인정보보호위원회**

### 주요 확인 사항

- 국외이전 유형별 요건 (제3자 제공 / 처리위탁 / 보관)
- 동의 vs 고지로 갈음 가능 여부
- 인증 제도 활용 (ISMS-P, CBPR 등)
- 개인정보처리방침 고지 사항

### 국외이전 방법

| 방법 | 요건 | 적합한 경우 |
|------|------|-------------|
| 정보주체 동의 | 별도 동의 취득 | 제3자 제공 |
| 고지 | 개인정보처리방침 공개 | 처리위탁, 보관 |
| 인증 | ISMS-P, CBPR 등 | 대량/반복적 이전 |
| 동등성 인정국 | 보호위 지정 국가 | 해당 국가로 이전 시 |

### 참고 사이트

- [개인정보보호위원회](https://www.pipc.go.kr/)
- [개인정보포털 - 국외이전 안내](https://www.privacy.go.kr/)

## 10. 국제뇌물방지

해외 프로젝트 수주, 현지 인허가 취득 등 외국공무원과 접촉이 있는 경우.

### 확인할 법령

```bash
fetch_law.py exact "국제상거래에 있어서 외국공무원에 대한 뇌물방지법"
```

### 관할 기관

- **검찰청** - 수사/기소
- **법무부**

### 주요 확인 사항

- 외국공무원 뇌물공여 시 **5년 이하 징역** 또는 벌금
- 국제상거래 관련 부정한 이익 목적 여부
- 에이전트/컨설턴트를 통한 간접 뇌물 위험
- 양벌규정 (법인도 처벌 대상)

### 관련 해외법 (참고)

| 법률 | 국가 | 특징 |
|------|------|------|
| FCPA | 미국 | 역외적용, 한국 기업도 적용 가능 |
| UK Bribery Act | 영국 | 가장 엄격, 적극적 방어의무 |
| Sapin II | 프랑스 | 기업 컴플라이언스 의무화 |

> **주의**: 위 해외법들은 한국 기업에도 역외적용될 수 있습니다.
> 해외 영업/입찰 시 현지 뇌물방지법도 함께 검토하세요.

### 참고 사이트

- [법무부 - 국제형사사법공조](https://www.moj.go.kr/)
- [OECD 뇌물방지협약](https://www.oecd.org/corruption/oecdantibriberyconvention.htm)

## 11. 기업결합 신고 (해외 M&A)

해외 기업을 인수하거나, 해외 기업과 합병하는 경우.

### 확인할 법령

```bash
fetch_law.py exact "독점규제 및 공정거래에 관한 법률"
fetch_law.py search "기업결합" --type admrul
```

### 신고 기관

- **공정거래위원회**

### 신고 대상

- 일방 자산/매출 **3,000억원** 이상 AND 타방 **300억원** 이상
- 대규모기업집단 소속 회사는 **사전신고 필수**

### 주요 확인 사항

- 신고 대상 여부 (자산/매출 기준)
- 사전신고 vs 사후신고
- 간이심사 해당 여부 (국내 시장 영향 없는 해외 M&A)
- 심사 기간 (일반 120일, 간이 15일)

### 참고 사이트

- [공정거래위원회 기업결합 신고](https://mna.ftc.go.kr/)
- [2025 기업결합신고 가이드북 (PDF)](https://mna.ftc.go.kr/resources/mna/file/pdfGuide2025.pdf)

## 12. 공급망 실사 (ESG)

EU 등 해외 시장에 일정 규모 이상 매출이 있는 경우.

> **참고**: EU CSDDD는 2027년부터 단계적 시행 예정입니다.
> 한국 기업도 직접 또는 간접(공급망) 적용 대상이 될 수 있습니다.

### 확인할 법령

```bash
fetch_law.py search "공급망" --type admrul
fetch_law.py search "ESG" --type admrul
```

### 주요 확인 사항

- EU CSDDD 직접 적용 대상 여부 (EU 매출 4.5억 유로 초과)
- 공급망 내 인권/환경 실사 의무
- 대기업 공급망에 속한 경우 간접 영향
- 위반 시 전 세계 매출 5% 이상 벌금 (EU)

### EU CSDDD 적용 일정 (예정)

| 시기 | 대상 |
|------|------|
| 2027년 | 직원 5,000명+, 매출 15억 유로+ |
| 2028년 | 직원 3,000명+, 매출 9억 유로+ |
| 2029년 | 직원 1,000명+, 매출 4.5억 유로+ |

### 참고 사이트

- [대한상의 EU 공급망 실사지침 가이드북](https://www.korcham.net/)
- [EU CSDDD 원문](https://eur-lex.europa.eu/)

## 관련 법령 더 찾아보기

위 가이드에서 다루지 않았지만, 상황에 따라 확인이 필요할 수 있는 법령들입니다.

### 업종별 추가 확인 법령

| 업종/상황 | 확인할 법령 | 비고 |
|-----------|-------------|------|
| 해외건설 | 해외건설촉진법 | 현지법인 설립 시 60일 내 국토부 신고 |
| 방위산업 | 방위사업법, 대외무역법 | 방산물자 수출 허가 |
| 금융업 | 금융회사의 해외진출에 관한 규정 | 금융위 사전신고/승인 |
| 원자력 | 원자력안전법 | 원자력 관련 기술/물자 수출 |
| 의약품 | 약사법 | 의약품 수출입 허가 |
